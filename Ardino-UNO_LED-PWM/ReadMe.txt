มี 5 Code นะครับ

1.Fixed		สามารถตั้งค่าความสว่างที่ต้องการได้เลยตั้งแต่  0 ถึง 255 (0=0% และ 255=100%)
2.Fade_in	ไฟจะค่อยๆสว่างขึ้นจนสุดแล้วจะวนลูปใหม่ไปเรื่อยๆ
3.Fade_out	ไฟจะเริ่มต้นด้วยความสว่างสูงสุดแล้วจะค่อยๆสว่างน้อยลงจนดับแล้วจะวนลูปใหม่ไปเรื่อยๆ
4.Fade_InOut	ไฟจะเริ่มจากดับแล้วจะค่อยๆสว่างขึ้นจนสุดแล้วจะค่อยๆดับลงวนลูปไปเรื่อยๆ
5.Fade_InOut_5	ไฟจะเริ่มจากดับแล้วจะค่อยๆสว่างขึ้นจนสุดแล้วจะค่อยๆดับลงวนลูปไปเรื่อยๆ พร้อมกัน 5 หลอด
