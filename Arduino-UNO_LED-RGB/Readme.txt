Arduino RGB LED

Youtube link : https://youtu.be/j_hGoY7csQE

มี 4 code
1.Basic ปรับทีละสีหรือปรับค่าสีตามที่ต้องการ ตั้งแต่ 0(ดับ) ถึง 255(สว่างสูงสุด)
2.Red_Green_Blue เป็นการแสดงค่าสี แดง เขียว น้ำเงิน วงซ้ำไปเรื่อยๆโดยแต่ละครั้งจะมีการหน่วงเวลา 0.5 วินาที
3.Function เป็นการนำโค้ดที่ 2 มาปรับปรุงโดยการสร้างฟังก์ชั่นเพื่อให้ง่ายต่อการเรียกใช้
4.Function_Set_Color เป็นการสร้างฟังก์ชั่นแบบที่เวลาเรียกใช้ต้องใส่ค่าสี RGB


*ค่าสีที่ออกมาจากหลอดอาจไม่ตรงกับค่าสี RGB จริง ถ้าต้องการค่าสีที่ถูกต้องแนะนำให้ปรับเทียบโดยการค่อยๆปรับค่าสีแดงเขียวน้ำเงินทีละสีให้ตรงกับที่ต้องการ
